Este script en Python sirve para hacer un mini análisis de red y del sistema, y está pensado para usarse en Windows o Linux. Lo que hace es escanear los puertos abiertos con nmap, checar qué servicios están corriendo (solo en Windows con PowerShell), y mostrar información de red (con ipconfig o ifconfig). Luego, guarda toda esa información en un archivo que puede ser .txt, .csv, .html o incluso .xlsx, dependiendo de lo que le pidas. Además, le calcula un hash MD5 al archivo para asegurarte que no fue modificado.

Todo esto se corre desde consola pasando el formato y el nombre del reporte como argumentos. El script está pensado para usarse como una herramienta rápida de diagnóstico o para hacer auditorías básicas de seguridad.
